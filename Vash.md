Vash short guide.
=================

- F keys
  - F1 Help
  - F2 Exit

- digits
  - 1 CLI (for return use ctrl-c or F2)
  - 2 Menu
  - 3 View
  - 4 Edit
  - 5 Copy
  - 6 Move
  - 7 Misc
  - 8 Remove
  - 9 Setup
  - 0 Exit

- keys
  - space Mark files
  - A Jump to vash library
  - L Execute ls -l
  - ~ cd home
  - : cd ..
  - F Execute file on current item
